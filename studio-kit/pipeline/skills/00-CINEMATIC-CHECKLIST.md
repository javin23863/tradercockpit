# 🎬 CINEMATIC SKILLS CHECKLIST — walk EVERY skill before you generate
_Go through these every time you make a video so output is cinematic. Each file compounds as you digest more references (`build-video-skills.cjs`) — the more you feed it, the sharper the library gets._

**MANDATE:** before generating any shot, open each skill below and apply its rules; after rendering, re-check the ones that failed. This is the studio's per-video craft gate (pairs with MONTAGE-CRAFT.md + STACK.md).

01. **[Idea & Concept](idea_concept.md)** — the core premise, scene-by-scene plan, why each shot earns its place
02. **[Characters](characters.md)** — character sheets, consistency, outfits, aging, diversity (avoid clone armies)
03. **[Props](props.md)** — prop sheets, depth/volume, controlling the exact action
04. **[Locations](locations.md)** — the location image is ~most of the video quality; angle + engine choice
05. **[Image Generation](image_generation.md)** — sheets, backgrounds, model-switching, quality checks before animating
06. **[Consistency & Continuity](consistency_continuity.md)** — lock characters/props/geography across shots — sheets, schematic maps, wet/dry variants, single-face, anchors
07. **[Camera Angle](camera_angle.md)** — framing that reads cinematic + gives the model depth
08. **[Camera Movement](camera_movement.md)** — how the camera moves so it feels shot, not generated
09. **[Realism & Detail](realism_detail.md)** — the texture/lighting cues that make it read as real, not AI
10. **[Prompting](prompting.md)** — how to write the generation prompt so the model nails it
11. **[Transitions & VFX](transitions_vfx.md)** — in-camera effects + scene-to-scene transitions with no editing cuts
12. **[Editing & Iteration](editing_iteration.md)** — the ad is the best seconds of 100 tries — cut on action, match cuts, pull keeper phases from many takes
13. **[Voiceover & Audio](voiceover_audio.md)** — narration/VO + sound that lifts the shot
14. **[Hook & Structure](hook_structure.md)** — how the VIDEO itself hooks + holds a viewer (every shot a hook)
15. **[Local & Cloud Generation](local_generation.md)** — two lanes to make every shot — cloud AI-gen for photoreal, local $0 engines (Hyperframes/Remotion) for MG + kinetic; pick per shot
