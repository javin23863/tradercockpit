# 🎬 SKILL — Transitions & VFX
_in-camera effects + scene-to-scene transitions with no editing cuts. Apply EVERY video. Rules compound as we study more references._

## from: Seedance 4K hyperreal AI short film breakdown (Adil) (2026-07-05T17:43:42.647Z)
- Put video-inside-video (footage playing on an in-scene TV) with zero VFX software: render the screen content separately, trim ~6 seconds, and attach it as a video reference in the room-scene prompt.
- Lock the on-screen display's size in the prompt so the referenced clip doesn't resize the TV between takes.
- Bridge scenes without an edit by making the final on-screen channel the opening shot of the next scene.
- Pull off scale transitions with text tweaks alone — no editing cuts — and let the generation carry the move.
- Stage big physics events (a truck smashing through barricades, a monster breaching from under snow like a shark) fully in-generation rather than compositing them.

## from: AI commercial/ad workflow (Adil) (2026-07-05T17:53:13.984Z)
- Build seam-free transitions with match cuts: match the opening action of each scene to the closing action of the previous — same hand, same motion — so it cuts clean with no visible edit.
- Generate in-camera VFX through prompt language: the body-rig snorricam with motion-blur background is a production-rig effect produced entirely by description.
- Construct fast insert montages (straight-down, worm's-eye, tight on the pour, the sip, cut fast) inside the prompt itself rather than in post.

## from: Hell Grind AI feature film — 28 pro tips (Higgsfield) (2026-07-05T17:56:56.911Z)
- If an effect element (e.g. a portal) looks cheap, regenerate the WHOLE element — don't patch around it.
- Build cut-to-cut sequences inside one multi-shot generation so lighting and positioning stay locked across the seam instead of jumping at an edit point.

## from: Reference clips — ultra-realism prompt structures (2026-07-05T18:02:10.692Z)
- Design hero shots around a single continuous impossible motion (object falling from sky to ground) that needs no cut — the motion itself is the transition.
- Use motion-tracking points in post to anchor any VFX layer to the footage so tracked elements inherit the camera's shake and breathing.
